# 프로젝트 컨텍스트: 부산 도시 안전 인프라 디지털 트윈

## 프로젝트 개요
- **주제**: 부산 도심의 교통·안전·공공 시설물을 3D 공간에 통합 시각화
- **목적**: 포트폴리오 / 공모전
- **핵심 기능**: 레이어 필터링 + 시설물 상세조회 + 위기레벨 색상 표시
- **지역**: 부산광역시

---

## 기술 스택
| 역할 | 도구 |
|---|---|
| 3D 시각화 (핵심) | Cesium JS (디지털 트윈 표준) 또는 deck.gl |
| 프론트엔드 | React + Tailwind CSS |
| 데이터 처리 | Python (geopandas, pandas) |
| GIS 전처리 | QGIS (좌표계 변환, 레이어 정리) |
| 지도 베이스 | OpenStreetMap + 부산 건물 3D 레이어 |

## 위기레벨 시스템 설계
격자(그리드) 단위로 위험도 점수를 산출해 색상으로 표시

- 🟢 **안전**: 시설 충분, CCTV 커버리지 양호
- 🟡 **주의**: 일부 사각지대 존재
- 🔴 **위험**: CCTV 공백 + 교차로 고밀도 + 어린이보호구역 겹침

**점수 산출 요소**
- CCTV 커버리지 공백 여부
- 교차로 밀도
- 어린이보호구역 중첩 여부

---

## 주요 기능 요구사항
1. **3D 지도 뷰** — Cesium 기반, 부산 실제 지형/건물 위에 레이어 표시
2. **레이어 필터링** — 카테고리별 on/off (교통, 방범, 시설물 등)
3. **시설물 상세조회** — 클릭 시 팝업으로 해당 시설 정보 표시
4. **위기레벨 오버레이** — 그리드 단위 색상 히트맵
5. **대시보드** — 카테고리별 시설물 수, 위험구역 현황 요약
